# ESG_preprocessing

This repository contains Python code developed to replicate the analysis procedures described in:

Nierula, B., Stephani, T., Bailey, E., Kaptan, M., Pohle, L.-M. G., Horn, U., Mouraux, A., Maess, B., Villringer, A., Curio, G., Nikulin, V. V., & Eippert, F. (2024).
A multichannel electrophysiological approach to noninvasively and precisely record human spinal cord activity.
PLOS Biology, 22(10), e3002828.
https://doi.org/10.1371/journal.pbio.3002828

The goal of this repository is to reproduce, step by step, the full data-analysis workflow applied to the publicly available dataset (ds004388, version 1.0.0)
